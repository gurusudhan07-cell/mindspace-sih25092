## About the Project

MindSpace is a 24/7 digital mental health support platform built exclusively for college students in India. It provides AI-powered emotional support, daily mood tracking, curated mental health resources, counselor appointment booking, and one-tap SOS crisis intervention.

---

## Features

- Daily Mood Check-in with emoji scale and journal entry
- AI Chatbot (MindBot) powered by Claude API for 24/7 emotional support
- Crisis keyword detection with automatic SOS trigger
- Resource Library with articles, exercises, and helplines
- Counselor Appointment Booking system
- SOS Crisis Button with national helplines on every screen
- Anonymous Mode for stigma-free access
- All data saved locally — progress never lost

## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/mindspace-sih25092.git

# Go into the project folder
cd mindspace-sih25092

# Install dependencies
npm install

# Start the app
npm start
```

Open http://localhost:3000 in your browser.
